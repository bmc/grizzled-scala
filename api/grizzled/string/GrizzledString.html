GrizzledString

class GrizzledString extends AnyRef

An analog to Scala's RichString class, providing some methods that neither RichString nor String provide. By importing the implicit conversion functions, you can use the methods in this class transparently from a String or RichStringobject.

import grizzled.string.implicits._
val s = "a  b          c"
println(s.tokenize) // prints: List(a, b, c)
